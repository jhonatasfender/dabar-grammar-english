**Regra editorial:** ao adicionar ou alterar um link no corpo de um artigo, atualize a linha correspondente nesta tabela (e os números de linha se o MDX mover).

**Gerado automaticamente** a partir do repositório; volte a gerar após grandes edições com:

```bash
npm run doc:cross-refs
```
