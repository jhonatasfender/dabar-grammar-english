## Core pattern after motion

_She flew to Boston **to** attend the workshop._ — purpose adjunct, not object of _fly_.

## _in order to_ / _so as to_

Add explicit **intention** or formality: _We pre-registered the analysis **in order to** limit researcher degrees of freedom._

## Negative purpose

_So as not to_ / _in order not to_: _He anonymised the logs **so as not to** identify participants._

## _for_ + NP / _for_ + _-ing_

When the subject of the purpose clause is **not** the matrix subject, _for_-phrases help: _The code is easy **for beginners to read**._ / _This tool is **for** processing batches._

## Five-minute practice

Take five purpose clauses; mark whether the **controlled subject** of the infinitive is the matrix subject; if not, try a _for_-remedy.
