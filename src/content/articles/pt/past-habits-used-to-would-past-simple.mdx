## _Used to_ + infinitivo

- Hábito: _We **used to** deploy on Fridays._
- Estado: _The API **used to** be synchronous._

Negação e perguntas usam **did** no inglês moderno: _We **didn’t use to** run lint in CI._

## _Would_ para eventos repetidos

- Iteração **dinâmica** em narrativa: _Every winter, the river **would** freeze._

Evite **_would_** com verbos **estativos** para descrição pura — prefira **used to** ou **passado simples**.

## Passado simples + advérbio de frequência

- _I **walked** to campus **most days**._ (hábito via **adverbiais**)

## Checklist rápido

1. O predicado é **dinâmico** o suficiente para **_would_** narrativo?
2. O hábito **contrasta** com o presente (**used to**)?
3. Evitaste **_would_** com **estados**?
