## Padrão central após movimento

_She flew to Boston **to** attend the workshop._ — adjunto de finalidade, não objeto de _fly_.

## _in order to_ / _so as to_

Acrescentam **intenção** explícita ou formalidade: _We pre-registered the analysis **in order to** limit researcher degrees of freedom._

## Finalidade negativa

_So as not to_ / _in order not to_: _He anonymised the logs **so as not to** identify participants._

## _for_ + SN / _for_ + _-ing_

Quando o sujeito da finalidade **não** é o sujeito matricial, frases com _for_ ajudam: _The code is easy **for beginners to read**._ / _This tool is **for** processing batches._

## Prática rápida (5 minutos)

Pegue cinco orações de finalidade; marque se o **sujeito controlado** do infinitivo é o sujeito matricial; se não, teste remendo com _for_.
