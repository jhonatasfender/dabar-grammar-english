## _used to + infinitivo_ (hábito/estado passado)

- _I **used to** smoke._ (já não)
- Negação: _I **didn’t use to** care._ / _I **used not to** care._ (mais raro)

Não há paralelo de presente _use to_ — para hábitos presentes use **usually**, **tend to** ou **will**.

## _be / get used to + nome / -ing_

- **Acostumado:** _She **is used to** long compile times._
- **Processo:** _We **are getting used to** the new linter rules._

O **to** é **preposição** — o complemento é **NP** ou **gerúndio**.

## Passiva _be used to + infinitivo_ (outra construção)

- _This modal **is used to** express permission._ (finalidade — **não** hábito)

## Checklist rápido

1. Depois de **used to**, vem **infinitivo simples**?
2. Depois de **be used to**, o complemento é **-ing** ou **nome** (sentido “acostumado”)?
3. Evitaste _use to_ de presente para hábitos?
