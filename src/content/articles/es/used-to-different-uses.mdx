## _used to + infinitivo_ (hábito/estado pasado)

- _I **used to** smoke._ (ya no)
- Negación: _I **didn’t use to** care._ / _I **used not to** care._ (más raro)

No hay paralelo de presente _use to_ — para hábitos presentes usa **usually**, **tend to** o **will**.

## _be / get used to + sustantivo / -ing_

- **Acostumbrado:** _She **is used to** long compile times._
- **Proceso:** _We **are getting used to** the new linter rules._

El **to** es **preposición** — el complemento es **NP** o **gerundio**.

## Pasiva _be used to + infinitivo_ (otra construcción)

- _This modal **is used to** express permission._ (finalidad — **no** hábito)

## Lista de revisión rápida

1. ¿Tras **used to** sigue un **infinitivo simple**?
2. ¿Tras **be used to**, el complemento es **-ing** o **nombre** (sentido “acostumbrado”)?
3. ¿Evitaste _use to_ de presente para hábitos?
