## like + SN

_She runs **like** the wind._

## like + gerundio (informal)

_Feels **like** **raining**._ — en prosa cuidada **as if**.

## as + SN (papel)

_She works **as** a mentor._

## Clausulas

- **as + finita:** _**As** we noted earlier, …_
- **as if / as though**

## Lista de revisión rápida

1. ¿**Función** (**as**) o **semejanza** (**like**)?
2. ¿**Cláusula**? **as if** en registro cuidado.
