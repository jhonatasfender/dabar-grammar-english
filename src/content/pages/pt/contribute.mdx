## Todos são bem-vindos

O **Dabar Grammar English** melhora quando leitores **corrigem erros**, **sugerem artigos**, **ajustam traduções** ou **abrem issues** quando algo não ficou claro. Não é obrigatório ser linguista profissional para contribuir.

## O que você pode fazer

- **Artigos novos ou revisões** (gramática, sintaxe, pragmática, inglês acadêmico, etc.).
- **Correções** em qualquer MDX nos idiomas `en`, `pt`, `es`.
- **Issues** no GitHub para bugs, links quebrados ou ideias.
- **Conversa antes de mudanças grandes** — se preferir, abra uma issue primeiro.

## Passos resumidos

1. Repositório: **[github.com/jhonatasfender/dabar-grammar-english](https://github.com/jhonatasfender/dabar-grammar-english)**.
2. Para bugs e sugestões: **[Issues](https://github.com/jhonatasfender/dabar-grammar-english/issues)**.
3. Regras detalhadas (frontmatter, fluxo de PR): **[CONTRIBUTING.md](https://github.com/jhonatasfender/dabar-grammar-english/blob/main/CONTRIBUTING.md)**.
4. Envie alterações por **pull request**; a revisão é feita pelos mantenedores.

Obrigado por ajudar a manter o estudo avançado de inglês **preciso, legível e multilíngue**.
