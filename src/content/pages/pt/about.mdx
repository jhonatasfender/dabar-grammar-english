## O que é este projeto

**Dabar Grammar English** é um site pensado para **leitura prolongada** de inglês em nível avançado — inclusive para quem cursa **Letras/Inglês** ou estudos equivalentes. Os textos longos ficam em **MDX** no repositório, com versões completas em **inglês, português e espanhol** (não usamos i18n de interface para o corpo dos artigos).

## O nome _dabar_

No hebraico bíblico, **דָּבָר** (_dabar_) costuma significar **palavra**, **discurso** ou **assunto** — algo dito que tem peso. O nome do projeto alude à **gramática como cuidado com o modo como as palavras funcionam**, sem reduzir a língua a exercícios mecânicos.

## Para quem é

- Estudantes avançados que querem textos com viés de **linguística** (gramática, pragmática, sintaxe).
- Quem estuda em mais de um idioma e prefere **artigos inteiros** por língua.
- Quem valoriza um layout **tipo livro**, confortável para ler no desktop ou no celular.
